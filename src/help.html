Draw

Line

  • Sebelum memulai menggambar, pilih menu pada bawah board
  • Pilih "line" dan "draw"
  • Pilih warna yang akan digambar
  • Tekan tombol “Ok” untuk mulai menggambar pada board
  • Buatlah garis pada canvas dengan melakukan mouse-click di dua titik

Square

  • Sebelum memulai menggambar, pilih menu pada bawah board
  • Pilih "square" dan "draw"
  • Pilih warna yang akan digambar
  • Tekan tombol “Ok” untuk mulai menggambar pada board
  • Buatlah persegi pada canvas dengan melakukan mouse-click di dua titik

Rectangle

  • Sebelum memulai menggambar, pilih menu pada bawah board
  • Pilih "rectangle" dan "draw"
  • Pilih warna yang akan digambar
  • Tekan tombol “Ok” untuk mulai menggambar pada board
  • Buatlah persegi panjang pada canvas dengan melakukan mouse-click di dua titik

Poligon

  • Sebelum memulai menggambar, pilih menu pada bawah board
  • Pilih "poligon" dan "draw"
  • Pilih warna yang akan digambar
  • Pilih "number of sides" untuk banyaknya titik poligon
  • Tekan tombol “Ok” untuk mulai menggambar pada board
  • Buatlah poligon pada canvas dengan melakukan mouse-click di beberapa titik
Color
  • Sebelum memulai mengganti warna, pilih menu pada bawah board
  • Pilih "color"
  • Pilih warna yang akan digambar
  • Tekan tombol “Ok” untuk mulai menggambar pada board
  • Klik bentuk yang ingin diubah warnanya
Move
  • Sebelum memulai memindahkan objek, pilih menu pada bawah board
  • Pilih "move"
  • Tekan tombol “Ok” untuk mulai menggerakkan titik pada board
  • Pilih titik dan geser
Resize
  • Sebelum memulai resize objek, pilih menu pada bawah board
  • Pilih "resize"
  • Tekan tombol “Ok” dan mulai memilih objek
  • Geser opsi "Resize(Square)"
Save
  • Pastikan gambar telah sesuai untuk disimpan
  • Tekan "save" pada bawah board
Load
  • Pastikan file bertipe JSON
  • Tekan "choose file" pada bawah board dan pilih file, otomatis gambar akan muncul
